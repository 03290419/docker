# Docker Network

컨테이너를 연결하기 위해 가상 네트워크를 만들고, 이 네트워크에 두 개의 컨테이너를 소속시킨다.

```shell
docker network create 네트워크_이름
```

```shell
docker network rm 네트워크_이름
```
