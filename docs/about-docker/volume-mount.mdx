# Volume and Mount

## 스토리지를 마운트하는 절차

스토리지 영역을 생성 -> 컨테이너를 생성(및 마운트)

볼륨 마운트는 볼륨 상위 커맨드를 사용해 먼저 볼륨을 생성한다.

```shell
docker volume create 볼륨_이름
```

```shell
docker volume rm 볼륨_이름
```

## 스토리지를 마운트하는 커맨드

바인드 마운트 커맨드

```shell
docker run (생략) -v 스토리지_실제_경로:컨테이너_마운트_경로 (생략)
```

볼륨 마운트 커맨드

```shell
docker run (생략) -v 볼륨_이름:컨테이너_마운트_경로 (생략)
```

## Example Code

```shell
 docker run --name apa000ex20 -d -p 8090:80 -v /Users/{USER}/Documents/apa_folder:/usr/local/apache2/htdocs httpd
```
